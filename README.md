# Ansible_Servers

## Пользовательские переменные
- Для каждого пользователя создайте отдельный файл в каталоге `vars/users/`, скопировав шаблон `vars/users/example.yml.example` и удалив расширение `.example` (например, `vars/users/ivan.yml`).
- Старый путь с `vars/ernestsh.yml` больше не используется и его можно удалить.
- Файлы в `vars/users/` автоматически подключаются плейбуком, поэтому достаточно добавить/обновить свой файл и запустить плей.

## Запуск
```
ansible-playbook -i hosts.ini playbook.yml
```
